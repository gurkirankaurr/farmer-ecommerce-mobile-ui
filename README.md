# 🌱 Farmer E-commerce Mobile UI

A mobile-first UI/UX design for a farmer-focused e-commerce platform aimed at simplifying agricultural product purchases.

## 📌 Overview

This project focuses on designing an intuitive and accessible mobile interface for farmers to browse, search, and purchase agricultural products. The goal is to create a seamless and user-friendly experience tailored for rural users.

## ✨ Features

* 📱 Mobile-first design approach
* 🛒 Complete user journey (Home → Product → Cart → Checkout)
* 🔍 Easy product browsing and search
* 🎯 Clean, simple, and intuitive UI
* 🔗 Interactive prototype built in Figma

## 🎯 Objective

To improve the digital shopping experience for farmers by providing a simple, accessible, and efficient interface.
